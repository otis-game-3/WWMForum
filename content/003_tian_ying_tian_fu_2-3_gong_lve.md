# 前言
2-3天賦：「無人死於**明王怒相**與**明王天怒**，**明王天怒**圓圈不相交」

![打法差異](/tian_ying_tian_fu_2-3_1.png)

先前8人打法因為輸出不足所以機制較多，以下說明為10人打法，理論上應該是打法二或打法三。（若輸出還是不足....就辛苦點用10人搞打法一）

無論何種打法，要達成天賦2-3的起點都是**金仗降魔**，只要畫面上方提示**金仗降魔**，所有人都要開始為達成天賦作準備。

# 站位
![二奶站位](/tian_ying_tian_fu_2-3_2.png)
若團隊為1承傷7輸出2治療，必然會有一名輸出要無柱子可站，此時盡可能在其他已站定的人之間找中心，確保圓圈不相交即可

兩名治療的群補應該能覆蓋承傷和大多數輸出，輸出若發現自已不在群補範圍時，在圓圈不相交的前提下自已移動進群補範圍；治療放完群補後發現有人不在群補範圍，盡可能指定施放**朝雨輕塵**避免意外。

承傷原則上就站在區域中心就不會圓圈相交。

![三奶站位](/tian_ying_tian_fu_2-3_3.png)
若團隊為1承傷6輸出3治療，每名治療顧2個輸出，應該是最穩的

# 其他機制處理方式

若採用打法一或打法二，有些機制的處理方式和一般打副本不同，單獨列出說明：

## 沙之分身 -> 金光轟擊

參考影片片段：[03:40-04:24](https://www.youtube.com/watch?v=MEBZ6XYDCu0&t=3m40s)

一般沙之分身的處理方式是拉到場邊再擊殺形成雕像，但在2-3天賦時建議在場中心擊殺，之後金光轟擊把雕像清掉，有助於後續處理2-3。

## 金仗降魔 -> 明王怒相 -> 佛光普照

參考影片片段：[04:25-05:10](https://www.youtube.com/watch?v=MEBZ6XYDCu0&t=4m25s)

明王怒相與明王天怒不同，明王天怒所有人頭上都會有田英的面具頭像，明王怒相只有少數幾人頭上有田英的面具頭像，只有頭上有頭像的人才需要進減傷區域。

但我看過不少影片明明是明王怒相，但所有人頭上都有田英的面具頭像，因此建議在達成2-3天賦時不論是明王怒相或是明王天怒，都比照明王天怒去處理，也就是**站六角形減傷區域**，**無人死亡**，**腳下圓圈不相交**。

# 天賦團開團流程

## 準備
1. 盡可能全員都有帶無相金身(承傷和治療必帶)
2. 開局前隊長按站位圖標註出金杖降魔要插杖的地方(紅色點)以及中心(承傷站位)
3. 開局前治療先講好第幾次明王怒相/明王天怒放無相金身，避免浪費，非接下來要放無相金身的治療可視情況協助輸出，例如治療1放第一次，治療放第二次，則開局到第一次之間治療2可放奇術協助輸出，第一次後到第二次則治療1可放奇術協助輸出
4. 承傷的無相金身適情況使用。

## 階段流程
1. 先練習弄出大範圍六角形減傷區域

    > 若失敗 -> 直接重來，並確認是否要調整標註點

2. 熟悉在明王天怒前走到對應位置

    > 輸出 -> 若點名插杖，直接到標記位置，然後就不動了，否則找附近的杖移動過去
    > 治療 -> 若點名插杖，直接到標記位置，插杖後找最近的治療位
    > 承傷 -> 若點名插杖，直接到標記位置，插杖後返回中心（若王就在中心，試試用凌雲踏飛過去）
